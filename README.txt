# InventoryTrackingApplication
 Scan QR / barcodes to track reception of material and material shipped 

Utilizes the following technologies:
MySQL Database, Java Spring Boot Server, Kotlin, React.js

Android appliation utilized to scan QR codes and barcodes of products contained within current inventory or which is expected to be received. Once a code is scanned takes action based on the information contained within to inform loader of key information on the inventory item used to facilitate faster and more accurate shipments and receptions of material. App then uploads information to a centralized MySQL database accessable through web server. Office staff can then download information from website on the load and reception as Excel files which can be used to conduct further analysis.
